# AI 自动化测试插件原型

## 1. 原型目标

本目录用于实现 AI 驱动的自动化测试全流程原型，验证从 PRD 解析、需求拆解、测试用例生成、接口自动化执行、测试报告生成到 Bug 单生成的完整闭环。

## 2. 核心流程

1. 读取 docs/PRD.md
2. 解析产品需求
3. 生成结构化需求
4. 生成测试用例
5. 执行接口自动化测试
6. 生成测试执行报告
7. 根据失败用例生成 Bug 单

## 3. 第一版实现范围

第一版先不做复杂 UI，只实现命令行原型。

优先支持：

- 读取 PRD 文档
- 读取测试用例
- 执行注册、登录接口测试
- 发现用户名小于 6 位仍可注册的预埋 Bug
- 生成测试报告和 Bug 单
